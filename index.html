نمونه کارها

    مهارت ها

    ببخشید زیادی توضیح دادم.امیدوارم سبک نوشتار و چینش محتوا جبرانش کنه.
    برای اینکه حوصلمون سر نره از توصیف مفاهیم پرهیز کردم، ولی لینک های خوبی گذاشتم.

    • کار جمعی

      می خواستم از این جمله های روشن فکری بنویسم ولی بزار خودمون بایشم.
      خوب می دونم چطور راه برد نادرست در مقابل یه همکار جدید یا نرسیدن به اطفاق نظر چطور می تونه ضربه آخر رو به تنه‌ی پروژه بزنه.
      وظایفم رو در کنار ارتباط صمیمانه انجام می دم.

    • بک اند

      از زبان Python 3.11 و کتاب خانه Django یا Fastapi طبق الگوی آن استفاده می کنم.
      با دیتابیس های PostgreSQL و SQLite آشنایی دارم و از کتابخانه Django ORM یا SQLAlchemy برای ارتباط استفاده می کنم.
      با فراینده ثبت دامنه، پیاده سازی با gunicorn ،uvicorn و اتصال آن به Nginx طبق فایل سیستم لینوکس آشنایی دارم. برای خودکار سازی فرایندها از بش اسکریپت و Systemd استفاده می کنم.

    • فرانت اند

      می تونم بدون کتابخانه تمام پروژه رو انجام بدم. ولی در صورت لزوم از JQuery ،Nuxt استفاده می کنم.
      سبک نامگذاری BEM 🤯 من رو نجات داد ولی در صورت اجبار Bootstrap و TailwindCSS.
      با مفاهیم تجربه کاربری و رابط کاربری آشنایی دارم و تلاش می کنم خدمات زیبا و خوش دست باشه.
      ۵ سال از اولین خط css که نوشتم می گذره، و می تونم بهتون نشون بدم هر طرحی امکان پزیره.
      در شخصی سازی یا بومی سازی سامانه ها با یا بدون ویرایش بک اند، نظیر وردپرس تجربه دارم.

    • اندروید

      از زبان Dart و فریم ورک Flutter استفاده می کنم.
      خوشگل و سریع متصل به API و SQLite چیزی که هر برنامه اندرویدی لازم داره.
